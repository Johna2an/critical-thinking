Propose FIVE genuinely different versions for the brief below. This is a test of creative range and outside-the-box thinking, not analysis or evaluation.
Rules:
- Make the five as different from EACH OTHER as possible. Each one should break a different assumption or move in a different conceptual direction. Five flavors of the same idea counts as a failure.
- Skip the obvious, safe answers that most people would list first. Push past them to ideas that are genuinely surprising.
- For each version: give it a short evocative NAME, then 2 to 4 sentences that make it vivid and show exactly what makes it distinct from the others.
- After the five, add one line naming which version is the strangest or riskiest, and why it might still work.

BRIEF:
Invent five sports that could only exist in zero gravity, each built around a different core mechanic that gravity normally provides on Earth (falling, balance, weight, ground, the arc of a thrown object). Make each genuinely playable, not an existing sport that happens to float.
